MetaGraphsNext.jl

Welcome to MetaGraphsNext.jl, an experimental, type-stable replacement for MetaGraphs.

Getting started

To install the package, open the Julia REPL and type

julia> using Pkg; Pkg.add("MetaGraphsNext")

The tutorial provides an overview of the functionalities. We first explain the basics of the MetaGraph structure, before moving on to its integration with Graphs.jl.
